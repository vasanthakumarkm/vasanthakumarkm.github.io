# Vasanthakumar KM — Portfolio Site

> **Live site:** [vasanthakumarkm.github.io](https://vasanthakumarkm.github.io)

A high-fidelity personal portfolio built to showcase 7+ years of enterprise digital transformation impact.

Built using **AI as a development accelerator** — the same way I scope and apply automation in transformation programmes. The content, strategy, structure, and all impact numbers are entirely my own. AI (Claude / Perplexity) was used to generate and iterate on the HTML, CSS, and JavaScript code, with me directing every design decision, reviewing every output, and deploying to GitHub Pages.

> 💡 This is how modern transformation leaders work: not by learning to code from scratch, but by knowing *what to build*, *why it matters*, and *which tools get you there fastest*.

## 🛠️ Tech Stack

- **HTML5 / CSS3** — custom properties, grid, clip-path animations, responsive design
- **Vanilla JavaScript** — no React, no Vue, no jQuery
- **GSAP 3 + ScrollTrigger** — scroll-driven reveals, counter animations, parallax watermarks
- **Canvas API** — live particle network in the hero section
- **Google Fonts** — Cabinet Grotesk · DM Sans · Instrument Serif
- **GitHub Pages** — zero-cost hosting, deployed directly from `main` branch

## 🤖 How AI Was Used

I want to be transparent about this, because I think it’s actually the more interesting story:

| What I Did | What AI Did |
|---|---|
| Defined the site structure, sections, and narrative | Generated HTML/CSS/JS code from my descriptions |
| Wrote all content, impact numbers, and career story | Suggested animation approaches and layout patterns |
| Made every design decision (colours, typography, spacing) | Produced code iterations I reviewed and approved |
| Debugged layout issues and directed all fixes | Refactored code on request |
| Deployed to GitHub Pages and managed the repo | None — I handled all Git operations |

The result: a production-quality site in days, not months — without a web development background.

That is not a workaround. That is the skill.
