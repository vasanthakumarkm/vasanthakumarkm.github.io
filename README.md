## 🛠️ Tech Stack

- **HTML5 / CSS3** — custom properties, grid, clip-path animations, responsive design
- **Vanilla JavaScript** — no React, no Vue, no jQuery
- **GSAP 3 + ScrollTrigger** — scroll-driven reveals, counter animations, parallax watermarks
- **Canvas API** — live particle network in the hero section
- **Google Fonts** — Cabinet Grotesk · DM Sans · Instrument Serif
- **GitHub Pages** — zero-cost hosting, deployed from `main` branch

---

## ⚡ Performance & Design Decisions

- **No build step** — single `index.html`, instant deploy, zero dependencies to break
- **Custom cursor** — gold dot + lagged ring, reinforces brand identity
- **Page curtain** — smooth load reveal so content never flashes unstyled
- **Dark/Light mode** — CSS variable swap, instant toggle, persists via `data-theme`
- **Mobile responsive** — grid collapses at 900px and 600px breakpoints
- **Magnetic buttons** — subtle cursor-tracking repulsion on CTAs
- **3D tilt cards** — perspective transform on hover across all card components
